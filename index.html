LINEA DE TIEMPO HTML Y CSS

Jhonnatan Julian Vanegas Rodriguez || Michael Stiven Sanchez Palma || Catalina Bernate Useche

1970

Se desarrolla el SGML (Standard Generalized Markup Language), un lenguaje de marcado que se convierte en la base para futuros lenguajes de etiquetado.

Como lenguaje proporciona un marco estructural que más tarde será adaptado para la web.

1980

Tim Berners-Lee, un físico del CERN (Organización Europea para la Investigación Nuclear), propone un sistema de "hipertexto" para compartir documentos entre científicos.

El concepto se convierte en la base para el desarrollo de HTML.

1989

Berners-Lee comienza a trabajar en la World Wide Web (WWW), un sistema de hipertexto avanzado para facilitar la comunicación y el intercambio de información entre científicos en el CERN.

Se llevo acabo la creación de un servidor y un navegador web que permiten el acceso y la compartición de información desde distintos ordenadores.

1991

Se publica el primer documento formal sobre HTML, titulado "HTML Tags". El documento describe las etiquetas básicas que constituyen la estructura de HTML.

1993

El IETF (Internet Engineering Task Force) propone que HTML se convierta en un estándar.

Aunque las primeras versiones (HTML y HTML+) no logran ser oficialmente reconocidas, el desarrollo continúa.

1994 - 1995

Mientras tanto, el W3C (World Wide Web Consortium) presenta dos propuestas para lenguajes de hojas de estilo: CHSS y SSP, que finalmente se unifican en CSS (Cascading Style Sheets), sentando las bases para la separación de contenido y presentación en la web.

1995

Se publica HTML 2.0, el primer estándar oficial de HTML, desarrollado por el IETF.

Aunque se denomina 2.0, es realmente el primer estándar oficial de HTML.

1996

El W3C asume la responsabilidad de la publicación de los estándares de HTML.

En diciembre de este año, se lanza CSS1, que se convierte en el estándar para definir colores, fondos e imágenes en páginas web.

Enero de 1997

Se publica HTML 3.2, la primera recomendación oficial del W3C. Esta versión incorpora avances como applets de Java y texto que fluye alrededor de imágenes.

Diciembre de 1997

Se lanza HTML 4.0, que incluye mejoras significativas como la integración de hojas de estilo CSS, mejorando la accesibilidad y los formularios en las páginas web.

1998

El W3C lanza CSS2, que introduce nuevas características como estilos para tablas y formularios, y mejora la eficiencia en la utilización del diseño en sitios web.

En abril de este año, se lanza también HTML 4.0, que refuerza la colaboración con CSS.

Diciembre de 1999

Se publica HTML 4.01, una revisión de HTML 4.0 que moderniza la manera en que se estructuran y presentan las páginas web.

Al mismo tiempo, se introduce CSS3, que añade módulos para animaciones, transformaciones y transiciones, proporcionando mayor dinamismo al diseño web.

2008

Se introduce HTML 5, marcando un avance significativo al hacer las páginas web más claras, rápidas y fáciles de mantener.

HTML 5 se enfoca en la adaptación multiplataforma, permitiendo que las páginas web se visualicen correctamente en dispositivos modernos.

2009

CSS Flexbox se introduce como un método para organizar elementos en filas o columnas, brindando mayor flexibilidad en el diseño adaptable a diferentes dispositivos.

2012

Se presenta CSS Grid, una función que permite diseñar layouts complejos mediante una cuadrícula de filas y columnas, facilitando la creación de estructuras más sofisticadas en las páginas web.

2014

HTML 5 se convierte en un estándar oficial, y aunque continúan las actualizaciones, los cambios son mínimos y fáciles de asimilar.

2021

Se emite HTML 5.3, con continuas actualizaciones por parte del W3C y WHATWG.

Esta versión sigue mejorando la interoperabilidad y la adaptabilidad de las páginas web, aunque sin cambios traumáticos en su estructura o funcionamiento.
